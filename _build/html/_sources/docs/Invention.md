# Invention

有一天早晨，我感觉有点没精神，不停地在看表。突然，听到旁边场地的教练对学生讲，要有信念，要坚持打完。虽然，我觉得在当时的情景下，用信念这个词有一点夸张，但是不知为什么，我也立刻来了精神，决定好好地打好每一个球。

过后想了想，语言，还真是一个矛盾的东西。有的时候，感觉语言很有局限性，有说不清楚的无力感，甚至会感觉越描越黑，倒真不如沉默是金。但是，有的时候，又觉得语言很强大，能穿越时空地带给人感受，能定格在人的记忆里。仔细想想，或许，这种矛盾才是合乎情理的，因为，毕竟每种语言都是人发明的。

不过，我倒是很好奇，有一天，人工智能会不会进化人类的语言？让语言变得更”智能“？
