# Context

说起理解，我曾经认为多多交流很重要。因为我觉得，通过只言片语，就想要理解别人，其实，不是一件容易的事情。

有一天聊天的时候说起语言。我说我感觉好几种语言听起来都很抑扬顿挫，有韵律感。然后一个女孩说，她感觉中文也是，我说哪有，我怎么感觉不到。

后来想了想，觉得，也许，一些刻在自己身上的东西，自己是完全感受不到的，别人告诉自己，自己也感受不到。

由此可见，交流，讲究的可能不是多少，也不仅仅是内容，情形的选择也很重要。太陌生的情形，感受得太肤浅。太熟悉的情形，又缺乏感受的空间。

所以，我觉得未来会有一个受欢迎的职业，context designer。不仅与AI交流的时候需要，人和人交流的时候也很需要。
