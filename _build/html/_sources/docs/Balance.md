# Balance

我一直都不大理解，为什么人们会说，成年人追求平静如水。追求快乐不好吗？

但是，今天，走在路上的时候，我好像突然能从某个角度理解一些了。任何一件事物，如果能带给人快乐，本质上可能是带来了一种能量。这种能量越大，当有一天这件事物消失的时候，对人的波动也会越大。

而每个成年人，或多或少，都会承担一些责任。小的波动，可能是给生活增添故事性。可是大的波动，可能就会波及一个人甚至更多的人的稳态。

这样想来，追求平静、避免大的波动倒真不失为一个好方法。不过，还有一个方法，就是把自己变成汪洋大海，让自己不执迷于一时的能量或波动。在每一个得与失之间，找到动态的平衡与平静。
